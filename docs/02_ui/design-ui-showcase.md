### MVP 簡化版

```
┌─────── Explorer (280px) ─────────┐ ┌─────── Chat/Preview (彈性) ───────────────────────────────────────┐
│ EXPLORER                         │ │                                                                  │
│                                 │ │ [檔案路徑]                                                       │
│ ▼ workspace                     │ │ t21-hello_world > s0-planning > c01-20240121_153000.chat.json   │
│   ▼ t21-hello_world 🏃         │ │                                                                  │
│     ▼ s0-planning              │ │ [內容區域]                                                       │
│       💬 c01-20240121_153000.. │ │ # 聊天檔案時：                                                   │
│       💬 c02-20240121_154500.. │ │ [User] 請按照需求編寫...                                         │
│                                │ │                                                                  │
│     ▼ s1-implementation        │ │ [AI] 我已分析完需求...                                           │
│       💬 c01-20240121_153000.. │ │                                                                  │
│       📄 navbar.v1.py          │ │ [User] 這部分需要調整...                                         │
│       📄 navbar.v2.py          │ │                                                                  │
│       📄 api-spec.md           │ │ [AI] 根據反饋，我建議...                                         │
│                                │ │                                                                  │
│     📄 task.json              │ │ ╭─────────────────────────╮                                     │
│                                │ │ │Write a message...       │                                     │
│   ► t20-feature_xyz           │ │ ╰─────────────────────────╯                                     │
│   ► t19-bug_fix               │ │ [📎附件] [發送 ➤]                                               │
│                                │ │                                                                  │
│                                │ │ # 一般檔案時：                                                   │
│                                │ │ [檔案內容預覽/編輯]                                              │
│                                │ │                                                                  │
└────────────────────────────────┘ └──────────────────────────────────────────────────────────────────┘
```

主要簡化：
- Explorer：
   - 保留基本的資料夾結構
   - 保留最基本的檔案類型圖示（💬聊天、📄一般檔案）
   - 只保留任務進行狀態（🏃）
   - 移除底部的視圖切換區

- 內容區域：
   - 統一的檔案路徑顯示
   - 根據檔案類型顯示對應介面（聊天或一般檔案）
   - 移除額外的操作按鈕，只保留基本的附件上傳和發送功能
   - 移除版本控制和協作功能
